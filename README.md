## Installation

Clone this repo from gitlab to your local machine:

```bash
  git clone https://gitlab.com/kysliakpavlo/dash-v2.git
```

Install dependencies:

```bash
  pnpm install
```

Run local server:

```bash
  pnpm dev
```

The server will running at port 3000 by default.
